葬爱就是一种非主流文化的常用词。是当今网络流行术语。且流行于非主流杀马特之中。葬，即埋葬，爱，即爱情，翻译成外语就是bury love 。

为什么要清空默认边距(外边距和内边距)？

——在企业开发中，为了更好的控制盒子的位置和计算盒子的宽高，在编写代码之前，一般先清除默认边距

如何清除默认边距？

①*{ margin: 0; padding: 0; } ←通配符选择器*会遍历当前页面中所有的标签，所以性能不好
②body,div,dl,dt,dd,ul,ol,li,h1,h2,h3,h4,h5,h6,pre,code,form,fieldset,legend,input,textarea,p,blockquote,th,td{margin:0;padding:0}
或者直接引入雅虎api的css ——企业开发中，经常使用这种来消除
